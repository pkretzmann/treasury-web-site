# The five journal types

- **Trade** — primary entry point for buying and selling securities. Doc Types: Purchase, Sale. Drives lot creation (Purchase opens a DVE lot) and lot consumption (Sale runs FIFO against open lots, posts realized P/L). The Market-Value Principle matters here because sale posting differs under Realization vs. Mark-to-Market.

- **Exch. Rate Adj. / Adjustment** — periodic mark-to-market revaluation (kursregulering). Doc Types: Price Adjustment, Currency Adjustment. Driven by the Adjust Security Prices report (which now creates one line per security-account with the exact-day fund price). Each posting closes prior outstanding unrealized DPLEs and records the new full delta from purchase price; G/L hits Inventory + Unrealized Profit accounts. Only valid for Mark-to-Market (or Both) posting groups.

- **Transfer** — moves a position between security accounts (depots) or between securities (e.g. fund mergers / splits). Doc Types: Transfer, Share Size Adj.. No P/L impact; the book value is preserved. Share Size Adj. is a sub-variant that rescales quantity × share size while keeping total value constant (1:10 stock split, etc.).

- **Return / Yield** — dividends and bond coupons. Doc Type: Yield. SLE Quantity is always 0 (the position doesn't move); Amount carries the net cash (Yield Before Tax − Dividend Tax). Income is recorded with negative sign on the SLE so it credits naturally in the G/L. Realization Type can be Realized (cash received) or Unrealized (accrued, later closed by the cash posting).

- **Correction** — manual fixups for entries that need adjustment but aren't a clean reversal. Doc Type: Reversal (plus other types when used to re-issue a corrected version). Apply Entry No. on the line points to the SLE to undo; the system mirrors SLE/DVLE/DPLE/SAcLE rows with negated amounts, closes the original, and re-runs the G/L posting with flipped signs. Subject to the LIFO reversal guardrails we just added.
