Fanebladet Generelt

Når du opretter en ny aktie, starter du på fanebladet Generelt. Her udfylder du grundoplysningerne — hvad aktien hedder, hvilken ISIN-kode den har, og hvilken valuta den handles i.

Felt Udfyldes Bemærkning
No. V/T Tildeles automatisk fra nummerserien, hvis denne er konfigureret i Værdipapiropsætning. Er nummerserien tom, skal nummeret angives manuelt.
Number Series T Udfyldes automatisk fra Værdipapiropsætning ved oprettelse. Kan ændres via assisteret redigering på No.-feltet.
Security Type T Vælg Stock (Aktie). Kan ikke ændres, når der eksisterer bogførte poster på værdipapir.
Issuer Code V Angiv udsteder. Feltet er valgfrit for aktier.
ISIN Code V Angiv eller vælg ISIN-koden for aktien. Oprettes automatisk som ny ISIN-kode, hvis koden ikke kendes i forvejen. Kan ikke ændres, når der eksisterer bogførte poster.
Name V Angiv aktiens navn. Udfyldes manuelt — der er ikke automatisk navnhentning fra ISIN i den nuværende version.
Search Name V Udfyldes automatisk på baggrund af Name, men kan redigeres manuelt. Vises kun via Vis mere.
Blocked V Markér for at spærre aktien. En spærret aktie kan hverken vælges på en kladdelinje eller bogføres.
Currency Code V Angiv valutakoden, hvis aktien handles i en anden valuta end regnskabsvalutaen. Lad feltet stå blankt, hvis aktien er noteret i regnskabsvalutaen (f.eks. DKK). Kan ikke ændres, når der eksisterer bogførte poster.
Country Code V Angiv landekode for aktien.
Remaining Quantity P Beregnes og opdateres automatisk på baggrund af bogførte poster.
Remaining Value P Beregnes og opdateres automatisk på baggrund af bogførte poster.

Undergruppe: Share Details

I denne undergruppe angiver du stykstørrelsen for aktien.

Felt Udfyldes Bemærkning
Share Size V Angiv aktiens stykstørrelse (nominel stykstørrelse pr. aktie). Feltet kan ikke redigeres, hvis der eksisterer åbne poster på aktien — ændringen skal i så fald foretages via Overførselskladden.
Forkortelser i kolonnen "Udfyldes": T = Tvungen (skal altid udfyldes), V = Valgfri (udfyldes efter behov), P = Programstyret (vedligeholdes automatisk, kan ikke redigeres).

Fanebladet Bogføring

På fanebladet Bogføring knytter du aktien til finanskontoplanen og definerer bogføringsgrupper og dimensioner. Disse oplysninger styrer, hvordan posteringer på aktien påvirker bogholderiet.

Felt Udfyldes Bemærkning
Posting Group T Angiv eller vælg den bogføringsgruppe, der knytter aktien til finanskontoplanen. Se vejledningen om bogføringsgrupper.
Global Dimension 1 Code T Angiv den primære dimensionsværdi (f.eks. afdeling). Feltet er obligatorisk, hvis dimensioner er konfigureret.
Global Dimension 2 Code T Angiv den sekundære dimensionsværdi. Feltet er obligatorisk, hvis dimensioner er konfigureret.
Report Grouping Method V Vælg grupperingsmetode til rapporter: Posting Group, Dimension Value eller Detailed information.
Report Grouping Code V Angiv den tilhørende gruppekode. Opslaget afhænger af metoden: bogføringsgruppe, dimensionsværdi eller detaljeret information.

Handlinger og links på aktiekortet

Fra aktiekortet kan du navigere til relaterede data og udføre handlinger:

  • Investment Fund Prices — se kurshistorikken for den tilknyttede ISIN-kode og følg udviklingen i aktiens værdi over tid.
  • Security Accounts — se hvilke depoter aktien er tilknyttet.
  • Comments — tilføj noter og kommentarer til aktien, fx for at dokumentere særlige aftaler eller observationer.

Praktisk vejledning til oprettelse

Hvad skal du altid udfylde?

Start med de felter, der er markeret med T (Tvungen):

  • Security Type: Vælg Stock.
  • Number Series: Udfyldes automatisk.
  • Posting Group: Knyt aktien til en bogføringsgruppe.
  • Global Dimension 1 Code og Global Dimension 2 Code: Fyld kun ud, hvis dimensioner er konfigureret i systemet.

Hvad kan du udfylde efter behov?

Felterne markeret med V (Valgfri) kan udfyldes efter behov:

  • ISIN Code: Hjælper med at identificere og linke til kurser.
  • Name og Search Name: Gør det lettere at søge og finde aktien senere.
  • Currency Code: Fyld ud, hvis aktien handles i en anden valuta end DKK.
  • Share Size: Angiv, hvis aktien har en bestemt stykstørrelse.

Hvad styres automatisk?

Felterne markeret med P (Programstyret) opdateres automatisk baseret på posteringer:

  • Remaining Quantity: Systemet beregner automatisk, hvor mange aktier du har tilbage.
  • Remaining Value: Systemet beregner automatisk værdien af det resterende beholdning.
  • Search Name: Oprettes automatisk fra Name, men kan ændres manuelt.
Vigtigt: Når du har bogført poster på en aktie, kan du ikke længere ændre Security Type, ISIN Code eller Currency Code. Plant disse oplysninger omhyggeligt fra starten.
